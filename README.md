# AI Automatic reply to Emails 

---

## Overview

This is just a simple workflow that Recieves the messages gotten in mails, then the message is then sent to the AI agent which then gets the labels in the Mail Account, assign the message to a particular label and then drafts a very nice and contructive message. which the user can send id he likes.
